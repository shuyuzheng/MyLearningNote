Biography

My name is Shuyu Zheng, a researcher in Oncology field. I studied in Clinical Medicine for 7 years and got the Master's Degree in 2017. When doing my project during postgraduate period, I was attracted by the world of bioinformatics, or data analysis. After graduation I have made an important decision in my career: Giving up Clinical Medicine and diving into Data analysis!

I started my journal in data analysis by learning basic mathematic subjects, like calculus, by myself. After getting basic R programming skill from online courses, I began to providing free data analysis service to to help some clinicians by doing some project on public clinical or genomic data.

In September, 2018 I got an opportunity to join in Network Pharmacology for Precision Medicine group in FIMM, University of Helsinki and eventually got touch in with the "real" data scientists. Although I didn't get any income from this "job", I have learned a lot in work. Now I'm developing an R package for the DrugComb Database (one of the main project of our group).

I really like what I'm doing now and desire to learn more about data analysis, machine learning, programming and so on.
